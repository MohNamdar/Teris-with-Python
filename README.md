Dependencies
------------

-  `Python > 3.8`
-  `Termcolor > 2.4`

Install
-------
```bash
$ git clone https://github.com/MohNamdar/Teris-with-Python
```
```bash
$ pip install termcolor
```

Usage
-----

1. Run Main.py with Python Interprator:

```bash
$ python main.py
```

2. Enjoy The Game
